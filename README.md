# EduChain - Blockchain-based Attendance App

EduChain is a **secure, cross-platform Flutter application** designed to manage student and teacher attendance using **biometric authentication** and **blockchain-ready architecture**. The app is adaptive for both Android and iOS platforms and provides role-based dashboards for students and teachers.

---

## 🌟 Features

- **Biometric Login**  
  Secure login using fingerprint or face recognition.

- **Role-based Dashboard**  
  - **Students**: View personal attendance records  
  - **Teachers**: Mark attendance and view aggregated attendance analytics

- **Blockchain-ready Registration**  
  Registration process generates biometric hash ready for blockchain storage.

- **Modern UI Design**  
  Adaptive Material & Cupertino widgets for Android and iOS.

- **Offline & Online Ready**  
  Easily extendable for future backend or blockchain integration.
